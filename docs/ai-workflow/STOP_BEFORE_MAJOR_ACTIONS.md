# Stop Before Major Actions

Before any major action, answer these questions.

## Stop if the change touches:
- image generation logic
- prompt assembly
- character identity
- child anchors
- companion anchors or sheets
- family coherence
- story-bank structure
- reader/book layout
- production generation flow
- payments
- QA gates
- fallback behavior
- style references
- anything that costs more than 1-2 test images

## Required questions
1. Is this a general system fix or a story-specific patch?
2. Could this break another story, child, companion, or style?
3. Does this affect production behavior?
4. Does this spend money?
5. What is the smallest safe validation run?
6. What should ChatGPT review before implementation?
7. What should Guy eyeball before continuing?

## Rule
If the answer is unclear: stop and ask.
