# Story Bank Matrix — 36 Stories MVP

## Overview

6 categories × 3 archetypes × 2 stories = **36 stories**

Each pair is designed for **maximum contrast** — different rhythm, energy, solution mode, and climax type.

Stories are generated in batches of 6 (one category at a time, all 3 archetypes).

---

## Solution Modes (must vary within each pair)

| Code | Meaning |
|------|---------|
| BUILDING | Child constructs/creates something physical |
| SACRIFICE | Child gives up something precious |
| RULE_DISCOVERY | Child figures out a hidden rule |
| HELPING | The "scary" thing actually needs help |
| CREATIVE_ACT | Child sings, draws, imagines — changes reality |
| CONFRONTATION | Child faces something directly, not by force but by standing ground |

## Generation Process (per batch of 6)

1. **I prepare** detailed story assignments (mechanical fields) for all 6 stories
2. **You send to ChatGPT** 3 stories at a time (one archetype)
3. **I QA** each story with the 4 binary checks
4. **Pass** → save to bank. **Fail** → specific fix instructions → regenerate
5. Repeat for second set of 3
6. All 6 done → commit → next batch

**Estimated pace:** ~2-3 sessions per batch → full 36 in 12-18 sessions.
